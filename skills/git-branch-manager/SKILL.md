# Git 分支管理技能

## 功能描述
此技能用于自动化处理 feature 分支到对应 test 分支的创建和同步流程。
当在 feature/A 分支上运行此技能时，它会自动创建或更新对应的 feature/test/A 分支。
特别适用于用户说出 **“合并测试”** 或 **“合并 test”** 等自然语言指令时调用此技能。

## 技能配置
name: git-branch-manager
description: 自动化管理 feature 分支到测试分支的映射和同步
category: git-tools
version: 1.0.0
author: global-config

## 输入参数
无特定输入参数，技能会自动识别当前分支并进行相应操作。

## 自然语言触发示例
- 当用户说：**“合并测试”**
- 当用户说：**“合并 test”**
模型在理解到用户想把当前 feature 分支合并/同步到对应测试分支时，应优先调用本技能。

## 执行流程
1. 识别当前分支名称
2. 确认当前分支是否为 feature/* 格式
3. 构建目标测试分支名称 (feature/test/*)
4. 检查目标分支是否存在
5. 如果不存在则创建新分支，否则合并当前分支到目标分支
